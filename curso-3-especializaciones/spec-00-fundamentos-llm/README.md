# Spec 00 — Fundamentos LLM para testers

**Prerequisito común de todas las especializaciones.** Una semana que paga todo el curso: cómo funciona un LLM por dentro (lo justo), cómo se le habla por API, y el experimento que te hará SENTIR por qué el testing cambia.

| # | Módulo | Resultado |
|---|--------|-----------|
| 1 | [El modelo y la API](modulo-01-llm-y-api.md) | Primer programa contra la API de Anthropic; intuición de tokens, contexto y prompting |
